This project was built using an **AI-Assisted Development** workflow. Rather than traditional manual coding, I acted as the **System Architect**, using **Cursor AI** to orchestrate the build process.

This project showcases a modern, responsive website for Kala Cafe, a cafe located at 141 Reclamation Street, Yau Ma Tei, Hong Kong. The site features a detailed menu, customer testimonials, contact form, and Google Maps integration.

### üß† The AI-Native Workflow
* **Rapid Prototyping:** I leveraged Cursor's **Composer** to scaffold the multi-page architecture (Home, Menu, Contact) in minutes, focusing my time on UX/UI design rather than boilerplate code.
* **Context-Aware Engineering:** Used the `@Codebase` feature to ensure CSS styling remained consistent across the entire project without manual repetition.
* **Iterative Refinement:** Every piece of AI-generated code was manually reviewed and refactored for performance, accessibility (A11y), and clean semantic HTML structure.
